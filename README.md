ColorsClassifier
===

Simple script that lets you order images based on color.

# Install

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install .
```

# Run

Make sure you have activated your venv first.

```bash
colorsclassifier --help
```
